# DevRitual

轻量级终端开发者仪式感与习惯养成引擎。

## 安装

```bash
pip install -e .
```

## 使用

```bash
devritual init                    # 初始化配置
devritual ritual create           # 创建仪式
devritual habit create            # 创建习惯
devritual stats                   # 查看统计
devritual dashboard               # TUI仪表盘
```

## 零外部依赖

本项目仅使用 Python 标准库，无需安装任何第三方包。
